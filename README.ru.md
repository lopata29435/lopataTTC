# Lopata

🇬🇧 [Read in English](README.md)

Десктопный клиент VPN-протокола
[TrustTunnel](https://github.com/TrustTunnel/TrustTunnelClient) для Windows,
macOS и Linux. Системный трей, профили серверов, импорт `tt://`-ссылок,
логи подключения, kill-switch, автообновления.

## Скачать

Установщик — в [последнем релизе](https://github.com/lopata29435/lopataTTC/releases/latest):

| Ваша система | Файл |
|---|---|
| **Windows 10/11** | `Lopata_*_x64-setup.exe` |
| Windows на ARM / 32-bit | `*_arm64-setup.exe` / `*_x86-setup.exe` |
| **macOS** (Intel и Apple Silicon) | `Lopata_*_universal.dmg` |
| **Ubuntu / Debian / Mint** | `*_amd64.deb` |
| Fedora / openSUSE | `*.x86_64.rpm` |
| Любой Linux (портативно, без установки) | `*_amd64.AppImage` |

Ядро VPN скачивается автоматически при первом запуске. Дальше и приложение,
и ядро обновляются сами — вручную ничего обновлять не нужно.

## Как пользоваться

1. Установите и запустите Lopata.
2. Добавьте сервер любым удобным способом:
   * кликните `tt://`-ссылку в браузере;
   * скопируйте `tt://`-ссылку и нажмите **Импортировать из буфера**;
   * импортируйте `.toml`-файл конфигурации.
3. Нажмите **Подключиться**.

Для создания VPN-интерфейса нужны права администратора: Windows спросит один
раз при запуске (окно UAC), Linux и macOS — при нажатии «Подключиться».

## Параметры профиля

| Параметр | Значения |
|---|---|
| Протокол | HTTP/2 (TCP) или HTTP/3 (QUIC) |
| Режим VPN | `general` — весь трафик через VPN, или `selective` — только указанные домены |
| Anti-DPI | маскирует VPN-трафик от глубокой инспекции пакетов |
| Kill-switch | блокирует трафик при обрыве VPN-соединения |
| Постквантовый TLS | вкл/выкл |
| Свой SNI, DNS-апстримы, исключения маршрутов | произвольно |
